# Structure by domain

<p class="lead mt-2 mb-4">Controller / Service / Repository — project structure, not URL paths.</p>

::left::

```text
src/app/
  orders/
    controller.py   # HTTP only
    service.py      # business logic
    repository.py   # data access
    dtos.py         # in/out shapes
    deps.py         # providers
  plugins.py        # app wiring
```

::right::

<div class="h-full flex flex-col justify-center">
<ul class="stars text-lg">
<li><strong>Folders map to your domain, not to URL prefixes</strong></li>
<li class="muted">Controller stays thin: parse, delegate, respond</li>
<li class="muted">Service is testable without HTTP; Repository hides the DB</li>
<li class="gold">Each layer is replaceable in isolation</li>
</ul>
</div>
