# Structure by domain (DDD)

::left::

```text
src/
  domain/
    orders/
      controllers.py  # HTTP only
      services.py     # business logic
      repositories.py # data access
      schemas.py      # in/out shapes
      dependencies.py # providers
      ...             # guards, signals, urls, etc
  config/             # settings & env
  db/                 # engine & migrations
  lib/                # shared utilities
  server/             # ASGI entrypoint
```

::right::

<div>
<ul class="stars text-lg">
<li><strong>Folders map to your domain, not to URL prefixes</strong></li>
<li class="muted">Controller stays thin: parse, delegate, respond</li>
<li class="muted">Service is testable without HTTP; Repository hides the DB</li>
<li class="gold">Each layer is replaceable in isolation</li>
</ul>
</div>
