# Advanced Alchemy - the CRUD you didn't write

::left::

```python
class Tag(UUIDv7AuditBase, SlugKey):
    __tablename__ = "tag"

    name: Mapped[str]
    description: Mapped[str | None]


class TagService(SQLAlchemyAsyncRepositoryService[Tag]):
    class Repo(SQLAlchemyAsyncSlugRepository[Tag]):
        model_type = Tag

    repository_type = Repo
    match_fields = ["name"]
```

<ul class="stars mt-3">
<li class="muted"><code>id</code> (UUIDv7), <code>created_at</code>, <code>updated_at</code>, <code>slug</code> - all from the base classes</li>
<li class="muted">Session &amp; transaction: plugin-provided, commits on successful response</li>
<li class="muted">Add <code>UniqueMixin</code> for <code>get_or_create</code> - idempotent inserts for lookup tables</li>
</ul>

::right::

```python
class TagController(Controller):
    path = "/api/tags"
    dependencies = create_service_dependencies(
        TagService,
        key="tags_service",
        filters={"search": "name",
                 "pagination_type": "limit_offset"},
    )
    @get(operation_id="ListTags")
    async def list_tags(
        self,
        tags_service: NamedDependency[TagService],
        filters: NamedDependency[
            SkipValidation[list[FilterTypes]]],
    ) -> OffsetPagination[Tag]:
        results, total = await tags_service.list_and_count(
            *filters)
        return tags_service.to_schema(
            results, total, filters, schema_type=Tag)
```

<ul class="stars mt-3">
<li class="muted">Service, filters &amp; pagination wired by <strong>one</strong> dependency call</li>
<li class="muted">Migrations included: <code>litestar database upgrade</code> (Alembic)</li>
</ul>
