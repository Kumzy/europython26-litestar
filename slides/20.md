# Common performance costs, made visible

<div class="leakfix mt-2">
  <div class="lf-row" v-click="1">
    <div class="leak"><span class="num-inline gold">1</span> <span class="muted">Work done on <em>every</em> request</span></div>
    <div class="arrow gold">→</div>
    <div class="fix">Scope it. Build once per app, not per request</div>
  </div>
  <div class="lf-row" v-click="1">
    <div class="leak"><span class="num-inline gold">2</span> <span class="muted">A blocking call inside an async handler</span></div>
    <div class="arrow gold">→</div>
    <div class="fix">Async all I/O or <code>sync_to_thread</code>, explicitly <span class="delta">one 2 s call = everyone waits 2 s</span></div>
  </div>
  <div class="lf-row" v-click="1">
    <div class="leak"><span class="num-inline gold">3</span> <span class="muted">Rebuilding heavy objects per request</span></div>
    <div class="arrow gold">→</div>
    <div class="fix">Match DI scope to real lifetime <span class="delta">reuse clients and pools across requests</span></div>
  </div>
  <div class="lf-row" v-click="1">
    <div class="leak"><span class="num-inline gold">4</span> <span class="muted">Serializing more than you return</span></div>
    <div class="arrow gold">→</div>
    <div class="fix">Tight DTOs, emit only the shape you declare <span class="delta">encode only the fields the client needs</span></div>
  </div>
  <div class="lf-row" v-click="1">
    <div class="leak"><span class="num-inline gold">5</span> <span class="muted">Global middleware on every route</span></div>
    <div class="arrow gold">→</div>
    <div class="fix">Attach it to the layer that needs it, not the app</div>
  </div>
</div>
